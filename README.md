# 🤖 AI Research Agent

> An AI-powered research assistant that autonomously searches the web, gathers information from multiple sources, reasons over the findings, and generates structured, citation-backed reports.

## ✨ Features

* 🔍 Autonomous web research
* 🤖 Multi-step AI reasoning
* 📚 Retrieval-Augmented Generation (RAG)
* 💬 Streaming AI chat interface
* 📄 Citation-backed responses
* 📝 Structured research reports
* 📂 Document upload & knowledge base
* ⚡ FastAPI backend + Next.js frontend
* 🐳 Docker support
* 🔐 Authentication & user management

# 🚀 Getting Started

## Clone the repository

```bash
git clone https://github.com/notAmecarmel/AI-Research-Agent.git
cd AI-Research-Agent
```

## Backend

```bash
cd backend
uv sync
make dev
```

## Frontend

```bash
cd frontend
npm install
npm run dev
```

## 🧠 How It Works

1. User submits a research query.
2. The AI plans the research strategy.
3. Relevant information is gathered from multiple sources.
4. Retrieved documents are indexed and searched using RAG.
5. The AI reasons over the collected information.
6. A structured report with citations is generated.
